# Results

This folder contains output plots generated by `src/eda.py` and `src/train.py`.

## EDA Plots

| File | Description |
|------|-------------|
| `constellation_distribution.png` | Bar chart of top 20 constellations by star count |
| `sky_distribution.png` | Scatter plot of all stars on the celestial sphere (RA vs Dec) |
| `magnitude_distributions.png` | Histograms of visual magnitude (Vmag) and absolute magnitude (AbsMag) |
| `spectral_class_distribution.png` | Bar chart of spectral classes by first letter |
| `correlation_matrix.png` | Heatmap of numeric feature correlations |
| `vmag_vs_distance.png` | Scatter plot of apparent brightness vs distance from Earth |

## Confusion Matrices

| File | Model |
|------|-------|
| `cm_knn.png` | k-Nearest Neighbours (k=7, Manhattan, distance-weighted) |
| `cm_rf.png` | Random Forest (600 estimators) |
| `cm_xgb.png` | XGBoost (baseline) |
| `cm_tuned_xgb.png` | Tuned XGBoost — **final model** |

## Model Performance Summary

| Model | Test Accuracy | Test Macro F1 | Notes |
|-------|--------------|---------------|-------|
| k-NN (k=7) | — | — | Baseline; distance-weighted + SMOTE |
| Random Forest | ~0.65 | ~0.69 | 600 trees, balanced class weights |
